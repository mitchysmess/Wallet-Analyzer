How it thinks

Wallet screen notes

  • Profitable means the wallet clears the configured minimums for activity, capital, win rate, realized PnL, and total PnL.
  • Borderline means there is some positive signal, but one or more core thresholds still miss.
  • Insufficient history means the wallet has too little trading history or deployed capital to judge reliably.

Birdeye wallet requests are paced automatically, and failed wallets are retried in extra passes. Lower Workers if you still see request errors on big lists.

Heuristics

How Token Intel scores wallets

  • Holder means the wallet appears in Birdeye’s token holder sample.
  • Early buyer is inferred from the earliest sampled token trades returned by Birdeye.
  • Cluster means multiple candidate wallets share the same first funding source.
  • Alpha score blends source tags, wallet profitability, holder size, and funding overlaps.

This is a practical wallet discovery tool, not perfect attribution. It does not claim a cluster belongs to a named person unless you later add your own wallet labels.
